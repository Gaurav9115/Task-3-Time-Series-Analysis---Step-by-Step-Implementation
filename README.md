# Task-3-Time-Series-Analysis---Step-by-Step-Implementation
Time Series Analysis on PJME Energy Consumption Dataset  This project performs Time Series Analysis and Forecasting on the PJME hourly energy consumption dataset using Python. The analysis includes data preprocessing, trend and seasonality decomposition, moving averages, and forecasting future energy demand using the SARIMAX model.
## Project Features

* Data cleaning and preprocessing
* Datetime conversion and indexing
* Time series visualization
* Daily and monthly resampling
* Trend and seasonality decomposition
* Moving average analysis
* SARIMAX forecasting model
* RMSE evaluation
* Forecast visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Statsmodels
* Scikit-learn

## Dataset

Dataset Columns:

* Datetime
* PJME_MW

The dataset contains hourly energy consumption data measured in Megawatts.

## Time Series Techniques Implemented

### 1. Data Resampling

* Hourly to Daily Average
* Hourly to Monthly Average

### 2. Time Series Decomposition

* Trend Analysis
* Seasonal Analysis
* Residual Analysis

### 3. Moving Averages

* 7-Day Moving Average
* 30-Day Moving Average

### 4. Forecasting

* SARIMAX Model
* Seasonal Forecasting
* 30-Day Future Prediction

### 5. Performance Evaluation

* RMSE (Root Mean Squared Error)

## Project Outcome

The project successfully identifies energy consumption trends and seasonal patterns and predicts future electricity demand using SARIMAX forecasting.
